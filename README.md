# [archived] Terraform MozReview module

A Terraform module to create a MozReview AWS stack

Input Variables
---------------
- `name` - Name of the MozReview stack

Modules Usage
-------------

```js
module "mozreview" {
  source = "github.com/dividehex/tf_mozreview"

  name = "MozReview"
}
```
